# 价格附件生成系统 - 功能清单

## 1. 项目概述

### 1.1 背景
用户需要根据Oracle数据库中的价格清单数据，按照价格模板规则匹配到不同的模板，并生成Word价格附件。

### 1.2 核心流程
```
输入(wybs报价单号) → Oracle查询 → 模板匹配 → Word生成 → 输出文件
```

### 1.3 输入输出
- **输入**: wybs（报价单号）- 通过API接口传入
- **输出**: Word文件 - 通过API接口返回下载

## 2. 功能模块

### 2.1 Oracle数据库连接模块
- [ ] **2.1.1** Oracle SDK集成 - 使用python-oracledb连接Oracle数据库
- [ ] **2.1.2** 数据库连接池管理 - 支持高并发
- [ ] **2.1.3** 连接配置管理 - 数据库连接参数外部配置

### 2.2 数据查询模块
- [ ] **2.2.1** 根据wybs查询报价单主表数据
- [ ] **2.2.2** 查询报价单明细数据（uf_htjgkst_dt1）
- [ ] **2.2.3** 数据字段映射 - 将Oracle字段转换为系统内部模型

### 2.3 模板匹配引擎
- [ ] **2.3.1** 模板规则加载 - 从Excel规则文件加载35个模板的匹配规则
- [ ] **2.3.2** 智能匹配逻辑 - 根据产品细分、定价组、是否集采、本年供货价类型字段进行匹配
- [ ] **2.3.3** 多模板支持 - 单个报价单可能匹配到多个模板，需分别生成

### 2.4 Word文档生成模块
- [ ] **2.4.1** 模板读取 - 读取docx模板文件
- [ ] **2.4.2** 数据填充 - 将价格数据填入Word表格
- [ ] **2.4.3** 多模板处理 - 根据匹配到的模板生成对应的Word文件

### 2.5 API接口模块
- [ ] **2.5.1** 生成价格附件接口 - 输入wybs，返回Word文件或zip包
- [ ] **2.5.2** 批量生成接口 - 支持批量传入wybs，生成多个文件并打包zip
- [ ] **2.5.3** 模板列表查询接口 - 查询可用的模板列表
- [ ] **2.5.4** 健康检查接口 - 服务状态检查

## 4. 非功能性需求

### 4.1 性能要求
- 支持高并发（10+并发请求）
- 单个文件生成时间 < 5秒
- 数据库连接池支持并发复用

### 4.2 错误处理
- 无效wybs返回明确错误信息
- Oracle连接异常处理
- 模板匹配失败处理
- Word生成异常处理

### 4.3 配置要求
- 模板文件路径可配置
- 规则文件路径可配置
- 数据库连接参数可配置

### 4.4 日志记录
- [ ] 记录每次生成的请求参数（wybs、时间）
- [ ] 记录模板匹配结果（匹配到的模板列表）
- [ ] 记录生成结果（成功/失败、文件路径）
- [ ] 记录异常信息（错误原因、堆栈）
