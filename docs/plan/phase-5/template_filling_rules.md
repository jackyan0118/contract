# 模板填写规则汇总
# ================

## 一、模板分类

根据表格列数和复杂度，43个模板分为4类：

| 分类 | 列数 | 数量 | 模板 | 特点 |
|------|------|------|------|------|
| 集采类 | 3列 | 16个 | 3.1-13.2, 9.1-10.2 | 序号、产品名称、供货价 |
| 酶免胶体金 | 6列 | 2个 | 模板1, 模板17 | 序号、产品类别、物料编码等 |
| 复杂类 | 7列 | 5个 | 模板2,5,8,11,14 | 有话术、规则复杂 |
| 外购类 | 8列 | 20个 | 模板15-16,18-35 | SAP代码、类别等 |

## 四、数据字段对应关系

### 4.1 明细表字段 (UF_HTJGKST_DT1)

| 字段名 | 说明 | 用于模板列 |
|--------|------|------------|
| WLDM | 物料代码 | 物料编码、SAP代码 |
| WLMS | 物料描述 | 品名、产品名称 |

| LSJ | 零售价 | 零售价 |
| GHJY | 供货价_元 | 供货价 |
| CPXF | 产品细分 | 产品类别、分类、类别 |
| LYXH | 物料生成来源 | (用于过滤) |
| DJZMC | 定价组名称 | (用于匹配) |
| SFJC | 是否集采 | (用于匹配) |

### 5.2 模板文件修改建议

为每个模板添加以下占位符：
1. 标题段落：`{{标题}}`
2. 金额单位：`{{金额单位}}`
3. 表格开始：`{{#明细表}}`
4. 表格结束：`{{/明细表}}`
5. 话术区域：`{{#话术}}...{{/话术}}`
